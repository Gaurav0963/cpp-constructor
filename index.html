C++ Lab: Understanding Constructors

Objective: Implement default, parameterized, and copy constructors in C++.

Problem 1: The `Book` Class

Problem Statement

Create a C++ class named `Book` to store details about books in a library, including title, author, and publication year.

Class Specification

The `Book` class should have the following private members:

  • title: A `std::string` for the book's title.
  • author: A `std::string` for the author's name.
  • year: An `int` for the publication year.

Required Public Members

  1. Default Constructor: Takes no arguments. Initializes `title` to "Untitled", `author` to "Unknown", and `year` to 0.
  2. Parameterized Constructor: Accepts `title`, `author`, and `year` to initialize the member variables.
  3. Copy Constructor: Takes a constant reference to another `Book` object and creates a copy. It should print a confirmation message.
  4. `display()` Method: Prints the details of the book.

Expected Output

--- Book 1 (Default Constructor) ---
Title: Untitled
Author: Unknown
Year: 0

--- Book 2 (Parameterized Constructor) ---
Title: The Hitchhiker's Guide to the Galaxy
Author: Douglas Adams
Year: 1979

--- Book 3 (Copy of Book 2) ---
// This line confirms the copy constructor was called
Book copy constructor called.
Title: The Hitchhiker's Guide to the Galaxy
Author: Douglas Adams
Year: 1979

Problem 2: The `Student` Class

Problem Statement

Design a C++ class named `Student` to hold basic information about a student, including their name, student ID, and GPA.

Class Specification

The `Student` class should have the following private members:

  • name: A `std::string` for the student's name.
  • studentID: An `int` for the student's ID.
  • gpa: A `double` for the student's GPA.

Required Public Members

  1. Default Constructor: Takes no arguments. Initializes `name` to "No Name", `studentID` to 0, and `gpa` to 0.0.
  2. Parameterized Constructor: Accepts `name`, `studentID`, and `gpa` to initialize the member variables.
  3. Copy Constructor: Takes a constant reference to another `Student` object and creates a copy, printing a confirmation message.
  4. `displayDetails()` Method: Prints the student's details.

Expected Output

--- Student 1 (Default Constructor) ---
Name: No Name
Student ID: 0
GPA: 0.0

--- Student 2 (Parameterized Constructor) ---
Name: Jane Doe
Student ID: 12345
GPA: 3.8

--- Student 3 (Copy of Student 2) ---
// This line confirms the copy constructor was called
Student copy constructor called.
Name: Jane Doe
Student ID: 12345
GPA: 3.8
